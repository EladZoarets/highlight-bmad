Use the Product, Architect, and Manager agents.

Goal:
Complete the full planning phase for the feature.

Steps:

1. Product
- Understand the assignment
- Define the intended behavior of the feature
- Identify inputs, outputs, and success criteria

2. Architect
- Design the minimal deterministic solution
- Define scoring rules
- Define ranking and tie-breaking
- Provide a small architecture description

3. Manager
- Break the solution into small executable tasks
- Order tasks logically
- Ensure tasks are small and implementable

Definition of Done:
- feature summary
- technical approach
- architecture description
- ordered task list
